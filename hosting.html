Software destacado

WordPress es una plataforma de publicación de vanguardia con un enfoque en la estética, los estándares web y la usabilidad.
Magento es una aplicación web de comercio electrónico de código abierto lanzada el 31 de marzo de 2008. Fue creada por Varien, basándose en componentes de Zend Framework.
Joomla es un sistema de gestión de contenido (CMS) galardonado, que le permite crear sitios web y potentes aplicaciones en línea.
OpenCart es un sistema de carrito de compras en línea de código abierto basado en PHP. Una sólida solución de comercio electrónico para comerciantes de Internet con la capacidad de crear su propio negocio en línea.
Drupal es una plataforma de código abierto y un sistema de gestión de contenido para crear sitios web dinámicos que ofrecen una amplia gama de funciones y servicios.
La solución de comercio electrónico PrestaShop se creó para aprovechar las innovaciones esenciales de la Web 2.0, como las funciones dinámicas impulsadas por AJAX y la ergonomía de última generación.
